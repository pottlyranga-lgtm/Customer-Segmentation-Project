# Customer Segmentation using K-Means Clustering

## 📌 Project Overview
This project uses the K-Means Clustering algorithm to segment customers based on their income and spending behavior. The goal is to identify distinct customer groups and gain insights that can help businesses create targeted marketing strategies and improve customer engagement.

## 🚀 Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Scaling
- Elbow Method for Optimal Clusters
- K-Means Clustering
- Customer Segment Visualization

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 📊 Workflow
1. Load and explore the dataset.
2. Clean and preprocess the data.
3. Perform EDA and visualize trends.
4. Apply feature scaling.
5. Determine the optimal number of clusters.
6. Train the K-Means model.
7. Visualize and analyze customer segments.

## 🎯 Results
The project successfully groups customers into meaningful segments, helping businesses understand customer behavior and make data-driven decisions.
